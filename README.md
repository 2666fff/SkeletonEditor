# SkeletonEditor
本程序是用来修改 spine的skel二进制文件, 无须回转json重新生成.
主要是修改 root节点的信息 包含:
X轴偏移 Y轴偏移 X和Y的缩放scale.
修改是在文件原生基础信息上修改的, 例 原始缩放为 3.0 填入 0.8 点击开始修改则缩放被改为 2.4

使用方法:
拖入文件夹或者文件,文件夹则修改指定名称.skel
